# 足球舆情系统 — 用户旅途说明

> 本文档与 PRD 对齐，经 2026-06-18 讨论修订。角色由充值金额决定，邀请码统一类型，系统通过 License 与舆情系统对接。

## 一、角色概览

| 角色 | 准入方式 | 可售额度 | License | 可生成邀请码 |
|------|---------|---------|---------|------------|
| 普通用户 | 免费注册 | 0 | ✗ | ✓ |
| 888 会员 | 充值 888 元 | 0 | ✓ | ✓ |
| 经销商 | 充值 5000 元 | 11 | ✓ | ✓ |
| 代理 | 充值 10000 元 | 22 | ✓ | ✓ |

- **角色由充值金额决定**，不由邀请码决定。
- 各充值独立，不互斥，不升级。888 会员可再充 5000 成为经销商（不退 888）。
- 代理/经销商充 5000/10000 时自带 License（即包含 888 会员权益）。
- 邀请码统一类型，一次性使用，用后失效。
- 收益关系仅限直接下级，不穿透更深层级。

---

## 二、邀请码机制

- **统一类型**，不区分目标角色（原"发展代理""发展经销商""888会员"三种合并为一种）。
- 一次性使用，用后失效。
- 只建立上下级关系，不影响注册者角色。
- 所有角色均可生成邀请码。

## 三、两个核心场景

### 场景 A：额度销售（不产生佣金）

代理/经销商消耗自身可售额度，为客户开通 888 会员。

```
代理/经销商（有可售额度）
    │
    ▼
录入客户手机号 → 短信验证
    │
    ▼
消耗 1 个可售额度
    │
    ▼
客户成为 888 会员 → 生成 License
    │
    ▼
客户的上级 = 销售者
    │
    ▼
✗ 不产生任何佣金（成本已在准入费中）
```

### 场景 B：推荐充值（产生佣金）

下级自己充值 888/5000/10000，系统记账直接上级的佣金。

```
下级自己充值
    │
    ├─ 888 元 → 成为 888 会员
    ├─ 5000 元 → 成为经销商
    └─ 10000 元 → 成为代理
    │
    ▼
系统记账直接上级佣金（首次奖励）
    │
    ▼
如适用，触发后续收益 / 长期奖励
```

## 四、普通用户旅途

```
注册/登录（无门槛，免费）
    │
    ▼
浏览系统功能
    │
    ▼
充值 888 元 → 成为 888 会员 → 获得 License
    │                              │
    ▼                              ▼
获得高级功能权益           每位用户限购 1 次
    │
    ▼
【可选】生成邀请码（统一类型）→ 分享给好友
    │
    ▼
好友通过邀请码注册
    │
    ▼
好友充值 888 元（场景 B）
    │
    ▼
获得推荐返佣：888 × 20% = 177.6 元
    │
    ▼
收益累积（提现通过工单申请）
```

### 关键节点

| 节点 | 说明 | 备注 |
|------|------|------|
| 注册 | 无准入门槛，无费用 | 需填写邀请码（可选） |
| 充值 888 | 888 元，限购 1 次 | 成为 888 会员，获得 License |
| 生成邀请码 | 统一类型，一次性使用 | 所有角色均可生成 |
| 推荐奖励 | 被推荐人充 888 后获得 177.6 元 | 仅限直接推荐的一级关系 |
| 推荐 5000/10000 | **不产生佣金** | 普通用户/888会员仅赚 888 推荐 |

## 七、佣金规则汇总

### 7.1 首次奖励（场景 B，下级充值时直接上级获得）

| 下级充值 | 上级是代理 (55%) | 上级是经销商 (40%) | 上级是普通用户/888会员 (20%) |
|---------|-----------------|------------------|--------------------------|
| 888 元 | 488.4 元 | 355.2 元 | 177.6 元 |
| 5000 元 | 2750 元 | 2000 元 | — |
| 10000 元 | 5500 元 | 4000 元 | — |

### 7.2 后续收益（仅代理→经销商关系）

- 代理的直接下级经销商，每次推荐他人充 888（场景 B），代理获得 **133.2 元/笔**。

### 7.3 长期奖励（定期结算，仅限直接下级全部收入）

| 上级角色 | 下级角色 | 长期奖励比例 |
|---------|---------|------------|
| 代理 | 代理 | 5% |
| 代理 | 经销商 | **不适用**（已由 133.2 元/笔替代） |
| 经销商 | 代理 | 4% |
| 经销商 | 经销商 | 4% |

## 八、系统对接

- 本系统与足球舆情分析系统通过 **License Code** 对接。
- 手机号做两系统用户关联，不做 SSO。
- 销售系统边界：注册 → 邀请码 → 角色管理 → 888会员开通 → 生成 License。
- 舆情系统边界：License 验证 → 高级功能展示。
- 两系统各自独立运行，各自独立登录。
